
# Hugo Language Switcher in Bootstrap

- Hugo language switcher test site. Language switcher lands user on the same page but in the chosen language (instead of directing user to homepage for that language)
- also tests the language capabilities of Pagefind search
- Thanks to [@jmooring](https://github.com/jmooring) for the initial code: https://discourse.gohugo.io/t/changing-languages-and-remaining-on-the-same-page/44031/3

- Incorporates:
  
1. Hugo https://gohugo.io/
2. Pagefind Search https://pagefind.app/

- To use:
  
1. install Hugo: https://gohugo.io/getting-started/quick-start/
2. Clone this repo
3. run: Hugo Server

- Demo Site:
https://jemmsuk.github.io/hugo-language-switcher/
